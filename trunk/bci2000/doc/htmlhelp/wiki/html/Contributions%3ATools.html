Tools

Contributions

There are the following tools available in the Contributions section:

  • Contributions:P300GUI: Optimize P300 classifier weights.
  • Contributions:Batch: A GUI for online system startup.
  • Contributions:BCIdiff: Analyze .dat files for differences.
  • Contributions:BCITimeFrequency: Data analysis in time and frequency domains.
  • Contributions:GenerateCalibration: Determine amplifier calibration parameters from a test signal recording.
