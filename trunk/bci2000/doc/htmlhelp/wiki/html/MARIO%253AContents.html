MARIO

MARIO is a collection of Matlab programs for off-line BCI2000 data analysis and prototyping of processing algorithms. The following information is available about MARIO:

A conceptual description of the MARIO software and its design.

Introduction to the use of the MARIO tool to analyze recorded sessions of mu rhythm modulation
