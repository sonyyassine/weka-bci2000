BCI2000 Help

BCI2000 is a general-purpose system for brain-computer interface (BCI) research. It can also be used for data acquisition, stimulus presentation, and brain monitoring applications.

This is the BCI2000 help start page. See the table of contents (to the left) for an overview of help topics.

When you are new to BCI2000, you might be interested in taking the introductory BCI2000 Tour.
